### Item Selection
| Shortcut | Action | Description |
|----------|--------|-------------|
| `↑` | **Select Previous** | Move selection up in cart |
| `↓` | **Select Next** | Move selection down in cart |

## ⚙️ CUSTOMIZATION

### How to Customize Shortcuts

1. **Open Settings:**
   - Press `Ctrl+,` OR
   - Click "Settings" in navigation menu

2. **Navigate to Keyboard Shortcuts:**
   - Click "⌨️ Keyboard Shortcuts" tab

3. **Reassign Shortcut:**
   - Click on shortcut you want to change
   - Press new key combination
   - System will detect conflicts
   - Confirm if no conflicts

4. **Reset to Defaults:**
   - Click "Reset All" button
   - Confirm reset

**⚠️ Conflict Detection:**
- System will warn if shortcut already in use
- Cannot override system shortcuts (F5, Ctrl+R, etc.)
- Must resolve conflict before saving

## 💡 POWER USER TIPS

### Combo Shortcuts
```
1. Type `3` → Press `F1` → Add 3x of F1 product
2. `Alt+F` → Type product → `Enter` → Instant add
3. `Shift+F` → Add to favorites → `F1` → Assign to key
4. `Ctrl+A` → `Delete` → Enter → Clear cart instantly
```

### Muscle Memory Training
1. **Week 1:** Print this reference, keep at desk
2. **Week 2:** Force yourself to use shortcuts even if slower
3. **Week 3:** Challenge yourself to full order without mouse
4. **Week 4:** Should feel natural, faster than mouse

### Common Workflows
**Fast Order (Known Customer):**
```
F1 → F2 → F3 → Ctrl+P → 1 → [amount] → Enter
(6 keypresses, 5 seconds)
```

**Search-Based Order:**
```
Alt+F → [product name] → Enter → Ctrl+P → 1 → 0 → Enter
(~8 keypresses, 8 seconds)
```
