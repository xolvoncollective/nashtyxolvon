### 1. 📱 WhatsApp Export for Reports

**Location:** Backoffice → Laporan → WhatsApp Button

**What It Does:**

### 2. 📅 Filter "Seminggu" (7 Days)

**Location:** Backoffice → Laporan → Filter buttons

**What It Does:**
- New filter option: "Seminggu" (Last 7 days)
- Shows data from 7 days ago until today
- Auto-calculate trends and averages

**Filters Available:**
1. **Hari Ini** - Today only
2. **Seminggu** - Last 7 days ✨ NEW!
3. **Minggu Ini** - This week (Sun-Today)
4. **Bulan Ini** - This month (1st-Today)

**How to Use:**
1. Buka Backoffice → Laporan
2. Click "Seminggu" filter button
3. Data untuk 7 hari terakhir ditampilkan
4. KPI, charts, dan tables updated automatically

**Data Shown:**
- Total Gross Sales (7 days)
- Total Net Sales (7 days)
- Total Orders (7 days)
- Average per day (Total / 7)
- Daily breakdown (per hari)
- Top products (aggregated)
- Payment methods (aggregated)

**Technical Details:**
- **Function:** `getDateRange(filter)`
- **Logic:** 
  ```javascript
  if (filter === 'Seminggu') {
    const sevenDaysAgo = new Date();
    sevenDaysAgo.setDate(sevenDaysAgo.getDate() - 7);
    from = format(sevenDaysAgo);
  }
  ```

**Payment Methods Displayed:**

**If columns don't exist, run migration:**
```sql
-- Add payment method columns if not exists
ALTER TABLE orders 
ADD COLUMN IF NOT EXISTS payment_gojek DECIMAL(10,2) DEFAULT 0,
ADD COLUMN IF NOT EXISTS payment_shopee DECIMAL(10,2) DEFAULT 0;

-- Update existing orders (if needed)
UPDATE orders 
SET payment_gojek = 0,
    payment_shopee = 0
WHERE payment_gojek IS NULL;
```

## 📞 Support & Troubleshooting

### Issue: WhatsApp button tidak muncul
**Solution:** 
- Refresh browser (Ctrl+R)
- Clear cache (Ctrl+Shift+Del)
- Check browser console for errors

### Issue: Payment Methods table kosong
**Solution:**
- Check if orders have payment_method field filled
- Verify database columns exist
- Check API response in Network tab

### Issue: Export message tidak lengkap
**Solution:**
- Ensure all data loaded before export
- Check reportData object in console
- Verify template function working
