Order Cards
Setiap kartu mewakili satu order. Chef interaksi dengan kartu ini.
🎴
Core
Informasi di Setiap Kartu
Semua info yang ditampilkan di setiap order card.
  • ✓ Nomor order (SNY-XXXX)
  • ✓ Nomor meja atau "TAKE AWAY"
  • ✓ Nama kasir yang input order
  • ✓ Live timer sejak order masuk
  • ✓ List item + qty + modifier
  • ✓ Customer notes / peringatan
  • ✓ Station badge per item (Bar, Grill, dll)
  • ⚙ Semua bisa toggle on/off di Settings
☑️
Core
Item Checklist
Chef bisa tick setiap item satu per satu saat selesai dimasak.
  • ✓ Tap item → item ter-coret
  • ✓ Progress bar bawah kartu mengisi
  • ✓ Tap lagi → uncheck
  • ✓ Saat semua item ✓ → tombol SIAP highlight
  • i Tidak wajib tick sebelum mark SIAP
🎨
Core
Warna Status Kartu
Warna border dan accent bar otomatis berubah sesuai status.
  • ✓ Orange — Order baru, normal
  • ✓ Kuning — Peringatan (5 menit)
  • ✓ Merah — Kritis + berkedip (10 menit)
  • ✓ Hijau — Sudah SIAP
  • ⚙ Batas menit bisa diubah di Settings
⬡
Stasiun & Filter
KDS bisa filter per stasiun dapur. Layar Bar hanya tampilkan item Bar.
🗂️
Settings
Stasiun yang Bisa Dikustomisasi
Semua stasiun bisa diubah dari Backoffice Settings.
  • ✓ Tambah stasiun baru
  • ✓ Ubah nama, emoji, warna
  • ✓ Atur urutan tab di KDS
  • ✓ Assign kategori menu → stasiun
  • ✓ Toggle aktif/nonaktif per stasiun
  • ✓ Set timer warning berbeda per stasiun
📺
Core
Multi-Screen Setup
Satu KDS per stasiun, atau satu KDS filter semua.
  • ✓ Filter "Semua" → tampilkan semua order
  • ✓ Filter "Bar" → hanya item Bar
  • ✓ Bisa buka beberapa tab browser
  • i Misal: 1 TV dapur + 1 iPad bar
  • i Semua realtime dari Firestore
⚡
Actions
Tombol aksi yang bisa dilakukan chef dari layar KDS.
✅
Core
Tombol SIAP
  • ✓ Tap → order status jadi "ready"
  • ✓ Kartu berubah hijau
  • ✓ POS kasir dapat notifikasi order siap
  • ✓ Sound alert (bisa di-toggle)
🍽️
Core
Tombol DISAJIKAN
  • ✓ Hanya muncul setelah status "ready"
  • ✓ Tap → order hilang dari grid
  • ✓ Counter "Selesai" hari ini naik
  • ✓ Data tersimpan di Firestore
➕
Core
Tombol + MASUK ORDER
  • ✓ Manual inject order (untuk testing)
  • i Di production order masuk otomatis dari POS
🔄
Core
Layout Toggle
  • ✓ Grid view — kartu besar, mudah dibaca dari jauh
  • ✓ List compact — banyak order muat di layar
  • ✓ Preferensi tersimpan per device
