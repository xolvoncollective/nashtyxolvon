# ⌨️ NASHTY OS - Keyboard Shortcuts Reference

**Quick Reference Card - Print and Keep at POS Station**

---

## 🧭 NAVIGATION

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+P` | **Payment** | Open payment dialog to complete order |
| `Ctrl+S` | **Save Draft** | Save current cart as draft for later |
| `Ctrl+N` | **New Order** | Clear cart and start new order |
| `Ctrl+D` | **Show Drafts** | Display list of saved draft orders |
| `Ctrl+H` | **Order History** | Open order history panel |
| `Alt+F` | **Focus Search** | Focus product search input |
| `Escape` | **Close/Cancel** | Close current dialog or cancel operation |

---

## 🛒 CART OPERATIONS

| Shortcut | Action | Description |
|----------|--------|-------------|
| `↑` `↓` | **Select Item** | Navigate up and down through cart items |
| `Delete` | **Remove Item** | Delete selected cart item (with confirmation) |
| `+` `-` | **Adjust Quantity** | Increase or decrease quantity by 1 |
| `Enter` | **Modifiers** | Open modifiers dialog for selected item |
| `Ctrl+A` | **Select All** | Select all items in cart |

---

## 🔢 QUANTITY ENTRY

| Shortcut | Action | Description |
|----------|--------|-------------|
| `0-9` | **Enter Quantity** | Type numbers before selecting product |
| `Escape` | **Clear Quantity** | Clear quantity indicator |

**Example**: Type `3` then click product → adds 3 units to cart
**Max**: 999 units per item

---

## 🎹 FUNCTION KEYS (Product Shortcuts)

| Shortcut | Action | Description |
|----------|--------|-------------|
| `F1-F12` | **Quick Add** | Add assigned product to cart instantly |
| `Shift+F1-F12` | **Assign Product** | Open product picker to assign to key |

**Setup**: Press `Shift+F1` to assign your best-selling product to F1 key

## 💡 TIPS & TRICKS

### Speed Up Order Entry
1. Assign top 12 products to F1-F12 keys
2. Use number keys for quantities: type `2` then press `F5`
3. Use `Alt+F` to quickly search for products
4. Use `+` `-` to adjust quantities without typing

### Keyboard-Only Workflow
1. `Alt+F` → Search product → `Enter`
2. Repeat for all items
3. Use `↑` `↓` to navigate cart
4. `+` `-` to adjust quantities
5. `Ctrl+P` to open payment
6. `Enter` to confirm payment

### Offline Mode
- Green badge = Online and synced
- Red badge with number = Offline with pending orders
- Orders are automatically synced when connection restored
- All shortcuts work offline

### Drafts & History
- `Ctrl+S` to save busy customer's order
- `Ctrl+D` to return to saved draft
- `Ctrl+H` to view order history
- `Ctrl+N` to start fresh order

## ⚙️ CUSTOMIZATION

All shortcuts can be customized in **Settings → Keyboard Shortcuts**

**Protected Shortcuts** (cannot be changed):
- `F5` (browser refresh)
- `Ctrl+R` (browser reload)
- `Escape` (close dialog)
- `Delete` (remove item)
- `↑` `↓` `+` `-` `Enter` (cart operations)
- `0-9` (quantity entry)

## 🚨 IMPORTANT NOTES

### Confirmations Required
These actions require confirmation to prevent accidents:
- Clear cart (`Ctrl+N`)
- Delete draft
- Remove all items (`Ctrl+A` + `Delete`)

### Smart Validations
- `Ctrl+P` (payment) only works when cart has items
- Shortcuts respect current UI state
- All actions logged for audit trail

### Accessibility
- All shortcuts have visible UI buttons as alternatives
- Mouse users can click instead of using keyboard
- Touch screen users can tap
- No functionality is keyboard-only

## 🆘 TROUBLESHOOTING

**Shortcut not working?**
- Check if dialog is open (close with `Escape` first)
- Verify shortcut hasn't been reassigned in settings
- Try clicking the button instead

**Quantity not applying?**
- Wait 5 seconds for indicator to clear
- Press `Escape` to clear manually
- Then type new quantity

**Function key not adding product?**
- Press `Shift+F1` to assign product first
- Check product is still available
- Product must be active and in stock
